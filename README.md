# 📡 Site Survey Tool for ISP App

A web-based application developed during my internship with **Infosys Springboard**, designed to assist Internet Service Providers (ISP) in conducting efficient site surveys and managing survey data.

---

## 🚀 Project Overview

The **Site Survey Tool for ISP App** helps ISP companies streamline the process of collecting, analyzing, and managing site survey data for network installation.

This application allows field engineers to:

* Record survey details
* Store location-based data
* Analyze feasibility for ISP setup
* Improve decision-making for network deployment

## ▶️ How to Run the Project

### Prerequisites:

* Java JDK installed
* Maven installed

### Steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/SiteSurveyToolForISPApp.git
   ```

2. Navigate to project folder:

   ```bash
   cd SiteSurveyToolForISPApp
   ```

3. Run the application:

   ```bash
   mvn spring-boot:run
   ```

4. Open browser:

   ```
   http://localhost:8080
   ```
